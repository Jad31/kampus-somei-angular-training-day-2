Système opérationnel

Bienvenue dans le projet d'apprentissage CodeQuest! Ce programme simule le système d'exploitation "Orion OS" pour une station spatiale. Vous allez construire ce système étape par étape à travers une série de missions qui couvrent les principaux concepts d'Angular.
